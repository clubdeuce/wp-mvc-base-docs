Class Base_Model_Metabox

The base metabox object model.

Example:

$sample_metabox = new Base_Model_Metabox(
        'sample_metabox',
        __( 'Sample Metabox', 'mytextdomain' ),
        'my_callback', //set this to null to use default plugin render_metabox callback
        'post',
        'normal',
        'default'
);
$sample_metabox->add();
